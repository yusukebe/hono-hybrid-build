# Hono hybrid build

Develop with Bun:

```txt
npm run dev
```

Build the static site with Bun:

```txt
npm run build:ssg
```

Deploy the app with SSG to Cloudflare Pages:

```txt
npm run deploy:ssg
```

Deploy the app with SSR to Cloudflare Workers:

```txt
npm run deploy:ssr
```
